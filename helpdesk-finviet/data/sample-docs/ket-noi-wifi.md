# Hướng dẫn kết nối WiFi văn phòng FinViet

## Thông tin WiFi

| Mạng | SSID | Mục đích |
|------|------|----------|
| Nhân viên | FinViet-Staff | Làm việc |
| Khách | FinViet-Guest | Khách đến thăm |
| IoT/Printer | FinViet-IoT | Thiết bị IoT, máy in |

## Kết nối WiFi nhân viên (FinViet-Staff)

### Windows
1. Click biểu tượng WiFi (góc phải dưới)
2. Chọn "FinViet-Staff"
3. Nhập mật khẩu WiFi (hỏi IT nếu chưa có)
4. Tick "Connect automatically"
5. Nếu hỏi Certificate → chọn "Connect"

### Mac
1. Click biểu tượng WiFi (góc phải trên)
2. Chọn "FinViet-Staff"
3. Nhập mật khẩu
4. Trust certificate nếu được hỏi

### Điện thoại
1. Settings → WiFi
2. Chọn "FinViet-Staff"
3. Nhập mật khẩu

## Xử lý lỗi WiFi

### Không kết nối được
1. Tắt WiFi → đợi 5 giây → bật lại
2. "Forget" mạng → kết nối lại
3. Kiểm tra đúng tầng (mỗi tầng có access point riêng)
4. Khởi động lại máy tính

### WiFi chậm
1. Kiểm tra có quá nhiều thiết bị kết nối
2. Thử ngồi gần access point hơn
3. Đóng các tab/app không cần thiết
4. Báo IT nếu tình trạng kéo dài

### Mất kết nối liên tục
1. Cập nhật driver WiFi
2. Kiểm tra tiết kiệm pin có tắt WiFi không: Device Manager → Network Adapter → Properties → Power Management → bỏ tick "Allow the computer to turn off this device"
3. Liên hệ IT nếu vẫn lỗi
